# README USAGE OF RandBreitWigner

## Approach 1

  ```javascript
    var gen = new BBT.JamesRandom( { seed: 12345 } );
    var n = 30;
    var A = [];
    var argu =  { size: n, vect: A, engine: gen, mean: 3, gamma: 0.5, cut: undefined };
    BBT.RandBreitWigner.ShootArray( argu );
    console.log(A);
    console.log( bw.FireM2() );
  ```
